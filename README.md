# Clash Verge Rev 多电脑一键配置统一与部署指南 (README)

本同步工具致力于帮助您将多台 Windows 电脑上的 **Clash Verge Rev** 客户端配置强制统一，使其拥有与您当前这台开发主机完全相同的极致分流规则、高性能 TUN 透明代理参数以及开发环回无污染防线。

为了严格响应您的安全防护需求，本同步包采用了 **“零订阅泄露 (Zero-Exposure)” 安全脱敏架构**：同步包中**没有任何您的订阅链接和节点数据文件**，完全保障您的账号隐私安全。您可以在任何其它电脑上放心部署。

---

## 📂 同步包结构说明

该 `ClashVergeSync` 文件夹包含以下文件，请确保它们处于同一目录中：

```text
ClashVergeSync/
├── ConfigBackup/
│   ├── verge.yaml          # 已脱敏的系统全局运行配置 (开启 TUN, 关闭系统代理, 锁定端口 7897)
│   └── Merge.yaml          # 精心设计的全局分流与网络覆写规则 (锁 MTU 1500, 瘦身防 DNS 污染)
├── Deploy-ClashVerge.ps1   # 核心 PowerShell 自动化安装与部署脚本
├── 双击一键统一部署.bat      # 极速一键部署引导批处理 (自动提升管理员权限)
└── README.md               # 本使用与维护指南文件
```

## ⚙️ 统一同步的配置项一览

本工具为您在新电脑上统一部署并锁定了以下核心高级参数，全方位保障您在 `powershell7`、`WSL2`、`Google`、`GPT`、`Antigravity`、`Codex` 中的网络畅通：

1. **自动识别并安装软件**：新电脑若未安装 Clash Verge Rev，脚本将调用 Windows 10/11 原生的包管理器 `winget` 自动下载并静默完成安装。
2. **强制启用 TUN 模式 (`enable_tun_mode: true`)**：使整个操作系统的物理和虚拟网络流量（包括 WSL 内部）均在网络层被透明代理自动接管。
3. **关闭系统代理 (`enable_system_proxy: false`)**：彻底排除本地系统代理，**从源头杜绝本地开发工具（如 agy CLI、Codex）在本地 127.0.0.1 上的进程间环回高频通信被代理误拦截造成的卡死**。
4. **锁定网络混合端口 (`verge_mixed_port: 7897`)**：保证所有本地终端的命令行及 API 连接端口绝对一致。
5. **锁定 Meta 内核 (`clash_core: verge-mihomo`)**：强制选用对新语法和虚拟网卡支持最好的 mihomo 核心。
6. **锁定网卡 MTU 为 1500 (`tun.mtu: 1500`)**：完美根除 WSL2 在镜像模式（`mirrored`）下继承大包导致 HTTPS 握手无限超时的致命物理丢包 Bug。
7. **极速瘦身 DNS 过滤器 (`dns.fake-ip-filter`)**：移除了海量海外域名，防止国内本地 DNS 污染；仅保留 `localhost` 等环回加速解析，实现出网流量的瞬间 Fake-IP 赋予与海外远端解析，网络成功率和速度极大飙升！

## 🚀 跨电脑极速部署三步走

请在您的其他电脑上执行以下简单步骤：

### 第一步：拷贝并双击运行
1. 将整个 `ClashVergeSync` 文件夹通过 U 盘、局域网共享或压缩包拷贝到您的其他电脑中。
2. 双击运行 **`双击一键统一部署.bat`**。
3. 系统若弹出“用户账户控制 (UAC)”提权请求，请点击 **“是 (Yes)”** 授予管理员权限。
4. 脚本会全自动运行：检测安装状态 -> 自动安装 -> 克隆覆盖 -> 自动热重启客户端。

### 第二步：在新电脑上手动加入您的订阅链接 (100% 隐私安全)
1. Clash Verge 自动重启拉起后，点击左侧菜单的 **“配置 (Profiles)”**。
2. 在顶部的地址栏中，贴入您自己的**订阅链接**，然后点击右侧的 **“导入 (Import)”**。
3. 导入成功后，鼠标左键选中该订阅文件（显示蓝色勾选）。

### 第三步：【至关重要】开启合并覆写模式 (Merge)
1. 在您刚导入的订阅卡片上，**鼠标右键点击它**。
2. 在弹出的菜单中，将配置合并模式选择为 **“Merge”**（或确保“合并”开关处于开启状态）。
3. 再次右键该订阅卡片，选择 **“刷新 / 重新加载 (Refresh)”**。
4. **至此，大功告成！** 我们的全局覆写规则（MTU 1500、分流路由、DNS 优化等）已完美与您的新节点数据无缝融合，开始畅享极致高速且稳定的开发网络！

## 🛡️ 安全防御与应急恢复常识

* **如何退回**：如果您想回滚原配置，本脚本非常贴心地在目标 AppData 目录下（`C:\Users\<您的用户名>\AppData\Roaming\io.github.clash-verge-rev.clash-verge-rev\`）生成了 `.bak` 备份文件。您只需将 `verge.yaml.bak` 重命名改回 `verge.yaml`，即可瞬间还原。
* **WSL 网络抖动一键修复**：如果您在新电脑上使用 WSL2 开发网络遇到临时卡顿，可直接双击我们为您随同步包放置在桌面的自愈工具，或者在普通的 Windows PowerShell 中执行 `wsl --shutdown` 强制重置 WSL 网络子系统即可秒级自愈。
